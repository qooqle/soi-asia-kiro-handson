# soi-asia-kiro-handson

## Purpose
This project was generated by Kiro as a hands-on demonstration of AI-assisted data analysis.
It explores internet adoption trends across 35 countries in Asia, Oceania, and the Pacific Rim from 2010 to 2024.

→ **[Read the full analysis report](ANALYSIS.md)**

## How to Start

### 1. Install Kiro

Download and install Kiro from [https://kiro.dev](https://kiro.dev).

### 2. Install Homebrew

Open Terminal.app and run the following command. Skip this step if you already have Homebrew installed.

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 3. Install uv

```bash
brew install uv
```

### 4. Open this repository in Kiro

Open this folder in Kiro via File → Open Folder.

### 5. Pin the Python version

Run the following command in Kiro's terminal.

```bash
uv python pin 3.12
```

This creates a `.python-version` file that locks the Python version for this project.

### 6. Install dependencies

```bash
uv sync --extra dev 2>/dev/null || uv pip install -r requirements.txt
```

### 7. Verify the setup

```bash
uv run python -c "import pandas; print('OK')"
```

If you see `OK`, you're all set.

### 8. Run the full pipeline

You can run the entire pipeline with a single command:

```bash
make all
```

Or run each step individually:

```bash
make acquire    # Download raw data from World Bank and ITU
make clean-data # Clean and build the panel dataset
make visualize  # Generate charts
```

If this works end-to-end, your environment is solid enough to handle most Python projects you build with Kiro.

Open the Kiro chat and start typing your prompts.
